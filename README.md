# Better Pitch

A single-page, horizontal-scroll parallax site that tells the story of how Pearmill landed Better.com — from silos to ecosystem. Internal team love letter, not a client-facing case study.

## Run it

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Edit the story

All chapter copy lives in a single array at the top of [`components/EcosystemTimeline.tsx`](components/EcosystemTimeline.tsx). Edit titles, bodies, pull quotes, deliverables, and stats there — no other file needs to change.

## Deploy

Push to GitHub, import into Vercel. No env vars required.
